Crowdhound Tutorial

A quick tutorial on how to add Crowdhound to an existing website.

Your Tutorial exercises will modify this ficticious site: snailadvisor.com.

As we proceed through the topics below, there is an example page for each topic, and also a link to a wiki page describing the code on the page. The topics build one on top of the next, gradually extending the functionality of the example.

At the end of each section there is an exercise, where you take the code from the example pages and apply it to the Snail Advisor site.

Part 14 - Advanced: Tenant Plugins

Crowdhound provides server-side extensibility through the use of "Tenant Plugins". The tenant plugins provide integration with external systems, and are responsible for user authentication and authorisation. For example, the user plugin might provide integration with Facebook and Twitter.
